# drTalk Platform Prototype

This repository contains an interactive, black-and-white prototype of the drTalk platform.

## Project Goal
Build a high-fidelity interactive prototype that demonstrates the onboarding flow and referral management system (Activity Center) as described in the mockups.

## Design Aesthetic
- **Black-and-White**: High contrast, wireframe-style UI.
- **Interactive**: Fully functional routing and state transitions for demonstration purposes.

## Getting Started
1. `npm install`
2. `npm run dev`
